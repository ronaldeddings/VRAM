# Caniuse Documentation

Welcome to the Caniuse browser compatibility database documentation. This comprehensive resource provides detailed information about web technology support across different browsers and versions.

---

## About Caniuse

Caniuse is the most comprehensive, up-to-date resource for browser compatibility information. It tracks support for CSS properties, JavaScript APIs, HTML elements, SVG features, and other web technologies across all major browsers and platforms.

### What This Database Provides

This documentation contains detailed compatibility data for:

- **CSS Features** - Modern CSS properties, at-rules, pseudo-classes, and selectors
- **JavaScript APIs** - Modern JavaScript language features and browser APIs
- **HTML5 Features** - HTML5 elements and attributes
- **SVG Support** - Scalable Vector Graphics features and elements
- **Media Formats** - Audio and video codec support
- **Browser Information** - Details about each major browser and its versions

Each feature includes:
- Support status for each browser version
- Known issues and limitations
- Vendor prefix requirements
- Standards specification links
- Use cases and implementation details

## How to Use This Documentation

1. **Search by Feature** - Start with the Features Index to find specific web technologies
2. **Check Browser Support** - Look up compatibility across all major browsers and versions
3. **Review Specifications** - Access official W3C and WHATWG specifications
4. **Understand Vendor Prefixes** - Find required vendor prefixes for experimental features
5. **Learn Best Practices** - Review use cases and implementation patterns

### Support Status Legend

- **Full Support** - Feature is fully supported in this browser version
- **Partial Support** - Feature is partially supported with known limitations
- **No Support** - Feature is not supported in this browser version
- **Polyfill Available** - Feature can be used with a JavaScript polyfill
