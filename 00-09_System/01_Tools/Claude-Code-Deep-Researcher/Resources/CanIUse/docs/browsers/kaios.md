# KaiOS Browser

## Overview

## Description

KaiOS Browser is the default web browser for KaiOS, an operating system designed for feature phones and entry-level smartphones. KaiOS is built on top of Firefox OS and provides web-based applications with a focus on emerging markets where smartphone penetration is lower.
